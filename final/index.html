WDD 231 Individual Project Plan

Movie Discovery Hub

A planning document for a movie recommendation website that helps visitors browse trending titles, explore categories, and save favorites for later.

Site Name

Movie Discovery Hub

This name was selected because it clearly explains the purpose of the site. It presents the website as a central place where users can discover movies, explore recommendations, and keep track of films they want to watch.

Optional domain: moviediscoveryhub.site

Site Purpose

The website will help users discover movies to watch based on categories such as trending, popular, and recommended titles. It will provide movie details including title, genre, release date, rating, and a short description. The site will also let users save favorite movies with local storage and open a modal dialog to view more detailed information without leaving the page.

Scenarios

  • Which trending movies should I watch this week?
  • Can I save my favorite movies and come back to them later?
  • Where can I quickly see the rating, genre, and release date for a movie?
